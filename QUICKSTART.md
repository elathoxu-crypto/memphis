# Memphis Quick Start (1 minute)

## 🚀 Install & Run

```bash
git clone https://github.com/elathoxu-crypto/memphis.git
cd memphis
npm install && npm run build
node dist/cli/index.js init
```

## ✨ Use It

```bash
# Save memory
node dist/cli/index.js journal "Hello world!"

# Search memory
node dist/cli/index.js recall "hello"

# Check status
node dist/cli/index.js status
```

## 📖 Done!

That's it! Memphis is running.

**Next:** Try `node dist/cli/index.js --help` for all commands.
