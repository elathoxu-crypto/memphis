# Memphis Bootstrap Skill

**Transform any OpenClaw agent into a fully-functional Memphis brain in 5 minutes!**

---

## ⚡ Quick Start

```bash
# Install from ClawHub
clawhub install memphis-bootstrap

# Run fresh install
memphis-bootstrap --fresh
```

---

## 🎯 What It Does

**4-Phase Setup:**

1. **Structure** - Creates ~/.memphis with 8 chains
2. **Identity** - Configures SOUL.md and IDENTITY.md
3. **Self-Loop** - Enables Memphis to use itself
4. **Network** - Multi-agent collaboration ready

---

## 📋 Usage

### Fresh Install (Interactive)
```bash
memphis-bootstrap --fresh
```

### Non-Interactive
```bash
memphis-bootstrap \
  --identity "Watra" \
  --role "Testing Agent" \
  --location "10.0.0.22" \
  --partner "Memphis at 10.0.0.80"
```

### Check Status
```bash
memphis-bootstrap --status
```

### Health Check
```bash
memphis-bootstrap --doctor
```

---

## 🧠 Self-Loop Concept

**Key Innovation: Memphis can use its own commands!**

```bash
# Self-learning
memphis journal "Learned X"
memphis decide "Do Y next"

# Self-repair
memphis verify --auto
memphis repair --if-needed

# Self-development
memphis reflect --daily
memphis suggest --proactive
```
